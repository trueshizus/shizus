# shizus.dev

## Overview

This is my personal website, where I share my projects and thoughts.

## Getting Started

1. Clone the repository
```bash
gh repo clone trueshizus/shizus
```

2. Install dependencies

```bash
cd shizus
yarn install
```

3. Run the development server

```bash
yarn dev
```

## Stack

- Next.js
- TailwindCSS
- TypeScript
- MDX
- Vercel
